## 遇到的问题

### mybatis-plus 实体驼峰问题

```
今天进行测试发现如果数据库内的字段带有"_"下划线，那么就查不出来对应的值，后来才知道mybatis-plus有驼峰检测，例如user_id在实体里就对应userId而不能用user_id，非常的非人类思维额
```

## mybatis-plus标识主键并自增

```java
//标识这个是表主键并自增，这样代码侵入性高，也可以换一种写法
@TableId(type = IdType.AUTO)
 private Integer userId;
```

```yaml
#这样式设置所有实体对应主键都自增
mybatis-plus:
  global-config:
    db-config:
      id-type: auto
```
